Identity Management Platform

We created a unified product platform that gives users and administrators the right access to the right apps and the right data.

Challenge

One UX for all

We designed a platform that performs for users of all ages, from grade-school students and administrators to business owners and their employees.

Solution

A flexible, intuitive framework

We simplified the platform’s structure to make it easier to manage software and data access across organizations.

Scope

  • Information Architecture
  • Interaction Design
  • Front End Development

Navigating without having to think

We designed an approach that anticipates and prioritizes based on what users need to do first, most, or almost never.

Unified under a common canvas

New modules and features don’t have to be built from scratch. A powerful underlying system of visual space can handle any combination of data that a user might need.

Engineered with an eye for detail

We built the front-end in EmberJS, working with the client to define API functionality. We built not just an app, but the first step in a broader platform to serve future customers. And, the resulting code makes the design shine to the fullest.

Bringing it to life

Beautiful animations and visual feedback—progress bars, transitions, and validated input fields—make users feel in control, without wondering whether something worked.
