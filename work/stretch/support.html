Stretch Support

The best backend for 
the best support team

Live Event Support System

We sat in the trenches with Stretch’s top-rated support team, and created a technology-powered workflow that doesn’t lose the human touch.

Challenge

Support Thousands of Live 
Events at the Same Time

Missing feeds, rain delays, audio glitches, bad focus -how can a system triage all of it?

Solution

Turn Humans into a 
Force Multiplier

Let automated analysis tools detect the obvious stuff, freeing the support team to manage each event - and each organization - personally.

Scope

  • Research
  • Information Architecture
  • Interaction Design

Fighting Fires

We created fast, effective views that let users instantly zoom in on any technical issue, with supporting tools and historical data to help them find the right solution.

Helping Customers Grow

Users can make sure every organization gets advice, support, and even inspiration on how to use the Stretch platform in creative ways.

In this together

We tailor-made an Issue system that keeps users engaged and focused, without stepping on each other’s toes.

Never skipping a beat

Historical analysis meets predictive modeling: our UX lets Stretch identify usage trends so that Stretch can grow alongside their customers.
