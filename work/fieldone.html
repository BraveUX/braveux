Asset Management Application

We made FieldOne’s asset management
platform easy to use and easy on the eyes.

Challenge

Make it easy to use.

FieldOne’s asset management suite needed a UX overhaul, including an easier interface with an modern design.

Solution

Revamp the UX.

The Brave team designed and built user interface for FieldOne’s core product, including a number of big time improvements for the end user.

Scope

  • Information Architecture
  • Interaction Design
  • Front-end Development

Tracking that makes sense and saves time.

The core users of the product, dispatchers, wanted a product that would make it easier to view and interact with data on the screen.

Easy to view and interact with data.

We created simplified views that show high-level availability of assets and resources.

Scheduling that works the way it should.

An improved “scheduler assistant” functionality relies on natural language and allows for a dispatcher set recurring appointments.

Finding what matters most.

We improved the information hierarchy so dispatchers can quickly identify and prioritize requests, tasks, and resources.

Functions that make sense.

We reconfigured the core controls into unified groupings, pairing similar functions to reduce the number of friction points for the user.

“Brave helped us turn our application from dated and clunky to modern and easy. Our customers love the changes, and so do we.”

Daniel Gittler, Director of Enterprise Services, FieldOne Systems
